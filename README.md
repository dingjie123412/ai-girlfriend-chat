# 🌙 深夜情感聊天 - AI女友

一个基于 Streamlit 的深夜情感聊天应用，与温柔知性的AI女友进行私密对话。

## ✨ 功能特点

- **多角色设定**：选择不同的AI女友角色（青梅竹马、大学白月光、咖啡店店员、游戏队友），每个角色都有独特的背景故事和性格特点
- **自定义人设**：支持自定义角色名字、背景故事和性格特点
- **渐进式对话**：对话会从普通聊天逐步深入到情感共鸣、暧昧聊天和私密话题
- **智能选项生成**：每次回复都会生成两个选项供你选择，让对话自然延续
- **防催睡机制**：AI不会主动劝你睡觉，会主动挽留继续聊天
- **话题库支持**：当对话陷入循环时，AI会主动切换新话题

## 🛠️ 技术栈

- Python 3.10+
- Streamlit
- DeepSeek API

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 配置 API Key

创建 `.env` 文件：

```env
DEEPSEEK_API_KEY=your_api_key_here
```

或者设置环境变量：

```bash
export DEEPSEEK_API_KEY=your_api_key_here
```

### 3. 运行应用

```bash
streamlit run chat.py
```

## 📝 使用说明

1. 在左侧边栏选择AI女友角色
2. 设置昵称、关系阶段和当前心情
3. 点击"开始聊天"按钮
4. 通过点击选项或输入自定义内容与AI女友聊天

## 🔒 隐私说明

- 聊天记录仅保存在本地浏览器会话中，刷新页面后会清空
- 不会保存任何聊天内容到服务器
- API Key 仅用于调用 DeepSeek API，不会上传到任何第三方服务
